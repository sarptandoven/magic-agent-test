# Timeline v1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the video timeline a persisted artifact that the UI and agent can inspect/edit so final videos do not end abruptly.

**Architecture:** Add a focused timeline module on the server that derives track clips from project state, applies bounded edit operations, and exposes restitch metadata. Persist the normalized timeline under `project_state.timeline`, surface it in frontend types, and upgrade the existing scene strip into a compact editor-style timeline with exact trim/final-hold controls. Agent tools wrap the same server timeline functions so user and agent edits share one contract.

**Tech Stack:** TypeScript, Fastify, Zod, @openai/agents tools, ffmpeg/ffprobe, Next.js App Router, React client components, Vitest, Node test runner for frontend helpers.
